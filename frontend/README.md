# NASA NEO Asteroid Dashboard – Frontend

This is the frontend for the NASA Near Earth Object (NEO) Asteroid Dashboard.

The application visualizes asteroid data from the NASA NEO API and displays information such as:

- Asteroid name
- Estimated diameter
- Potentially hazardous status
- Close approach date
- Relative velocity
- Miss distance from Earth

The frontend is built using React and communicates with the backend API to retrieve asteroid data.

## Tech Stack

React  
JavaScript  
CSS  
Axios

## Running the frontend locally

Navigate to the frontend folder:

npm install  
npm start

The application will run on:

http://localhost:3000

## Project Structure

src/
components/
services/
styles/

## Backend API

The frontend communicates with the Node.js backend service which processes and provides NASA asteroid data.
